# Overview

MLAG Vision is an open-source scientific platform dedicated to the **geometric analysis of organized forms**.

The first application domain is the analysis of crop circles (agroglyphs), but the software is designed as a generic geometric engine that can also be applied to:

- Crop circles
- Sacred geometry
- Gothic rose windows
- Mandalas
- Archaeological plans
- Historical architecture
- Technical drawings
- Symbols and logos
- Martial arts movement diagrams

MLAG Vision does **not** attempt to determine the origin or meaning of a figure.

Its purpose is to:

- observe
- reconstruct
- measure
- compare
- classify

using reproducible geometric methods.

---

# Project Philosophy

MLAG Vision follows one fundamental principle:

> **Every complex figure is the result of a geometric construction.**

The software therefore focuses on recovering that construction.

The workflow is always:

Image

↓

Geometry

↓

Reconstruction

↓

Graph

↓

Analysis

↓

Report

# Main Features

## Geometry Engine

- Point
- Line
- Segment
- Circle
- Arc
- Polygon
- Spiral
- Tangency
- Intersection
- Symmetry

---

## Computer Vision

- Perspective correction
- Noise reduction
- Edge detection
- Circle detection
- Line detection
- Arc detection

Powered by OpenCV.

---

## Reconstruction Engine

Automatic reconstruction of:

- geometric centers
- radii
- axes
- tangencies
- construction hierarchy

---

## Graph Engine

Each figure is transformed into a geometric graph.

Nodes represent:

- centers
- tangencies
- intersections

Edges represent geometric relations.

This allows structural comparison between figures independently of their visual appearance.

---

## MLAG Classification

Automatic computation of:

- complexity
- symmetry
- tangencies
- construction hierarchy
- geometric family

Generation of the MLAG Passport.

---

## Artificial Intelligence

MLAG AI does **not** learn directly from images.

Instead it learns from reconstructed geometric graphs.

This approach improves explainability and reduces bias.
